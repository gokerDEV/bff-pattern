## Documentation Scope

These documents describe the system architecture, runtime boundaries, data flows, authentication model, generated API client structure, deployment model, and enforceable architecture rules.

Process notes, discovery drafts, and review working notes are intentionally kept out of this folder so the repository documentation stays focused on the actual template architecture.
